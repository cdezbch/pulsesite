Endpoint Operations Platform

A single desktop console for an internal device-operations team — monitor device stations, look up devices, run reports, track OS compliance, and email overdue-device users, all in one app.

🪟 Windows desktop app ⟳ Self-updating MDM-connected Live reporting Apple OS tracking
What it does

Five workspaces, one window

Each tab is a focused tool. They share one device pull where it makes sense, so the data stays consistent and fast.

📊 Dashboards

Live, at-a-glance summary cards fed from your device-management systems.

  • Checkout by building, overdue (30/60/90 bands), retired count, device inventory
  • iOS Compliance card — % of the fleet on the latest release
  • Drag cards to rearrange; swap any slot for a report via the ▾ menu
  • Per-card CSV, plus one-click full snapshot to a multi-sheet Excel
  • Optional auto-refresh; results cached so they show instantly on launch

🔎 Device Lookup

Search the managed device fleet by serial, or list every device.

  • Key managed-app versions per device
  • Active vs. inactive (enrolled vs. unenrolled / wiped), color-coded
  • Organization / BYOD filters, favorites, and saved filter presets
  • CSV export of results or the whole fleet

📱 iOS Version Filter

A Device-Lookup view built for OS compliance.

  • Filter by version with =, <, or > (e.g. “behind 18.x”)
  • One-click “Behind latest” against the live Apple release feed
  • Compliance readout: “X of Y current (NN%)”
  • Sortable by real version order; CSV export

📡 Station Monitor

Reachability monitoring for device stations from a CSV list.

  • Online / Offline / Pending status with live rescans
  • Clickable summary cards filter the table instantly
  • Scheduled auto-scan, with optional offline-device email alerts

📈 Reports

Live reports pulled straight from the device-platform API.

  • 30 Days Overdue, Wrong Dock, Retired, Daily Checkout, Overdue by Building
  • Color-coded overdue bands; styled Excel export
  • Plus a Check-In Status view from a reporting export

✉️ Send Overdue Email

The 30-day overdue escalation, handled by hand when needed.

  • One personalized email per device, sent individually (effectively BCC)
  • Pick All / 30 / 60 / 90-day bands, individuals, or “previously emailed”
  • Re-verify still-overdue + confirm/preview before anything sends
  • Editable template, a per-batch summary email, and a local send history
  • Daily trend line + an optional tab badge of the emailable count
Always in view

Latest Apple OS, in the header

Pulse watches the newest iOS, iPadOS and macOS releases for the devices the team tracks (via a public Apple release feed) and surfaces them as pills on every tab — turning green and alerting you when a new version drops.
